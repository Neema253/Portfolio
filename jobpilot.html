JobPilot

L'idée du projet

  • Projet fil rouge BTS SIO SLAM sur 6 séances
  • Mise en relation candidats et entreprises
  • Les entreprises publient des offres d'emploi
  • Les candidats consultent et postulent aux offres
  • Authentification requise pour les actions CRUD
  • Consultation publique sans connexion possible
🏗️

Architecture MVC

  • OffreController — CRUD + filtres + pagination
  • CandidatController — CRUD + pagination
  • EntrepriseController — CRUD + pagination
  • CandidatureController — postuler / retirer
  • Routes publiques et protégées par middleware auth
  • Relations Eloquent : hasMany & belongsToMany

Routage

Les routes REST de l'application.

Les routes index et show sont publiques — toutes les actions d'écriture sont protégées par le middleware auth.

MéthodeRouteActionAccès
GET/offresListe paginée + filtresPublic
GET/offres/{id}Détail + candidaturesPublic
POST/offresCréer une offreAuth
PUT/offres/{id}Modifier une offreAuth
DELETE/offres/{id}Supprimer une offreAuth
POST/candidaturesPostuler à une offreAuth
DELETE/candidatures/{id}Retirer sa candidatureAuth
Le même pattern s'applique aux routes /candidats et /entreprises — CRUD complet avec pagination conservant les filtres via withQueryString().

Points clés

Quelques exemples.

🔗
Relation many-to-many
Un candidat peut postuler à plusieurs offres et une offre reçoit plusieurs candidatures. Géré via une table pivot candidat_offre et les méthodes Eloquent attach/detach.
🔍
Filtres dynamiques
Les checkboxes soumettent automatiquement un formulaire GET. Le controller filtre avec whereIn('type_contrat', $request->types) et la pagination conserve les filtres via withQueryString().
🛡️
Protection des routes
Les routes index et show sont publiques. Les actions create, store, edit, update, destroy sont protégées — redirection automatique vers /login si non connecté.
⚡
Flash messages globaux
Les messages de succès et d'erreur sont affichés dans le layout principal, évitant toute duplication dans les vues individuelles. Ils disparaissent à la prochaine navigation.
